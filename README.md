# Job Application Tracker (HTML, CSS, JS)

A simple web application to track job applications with features to add, edit and delete entries.  
Built using only HTML, CSS and vanilla JavaScript.

## Features

- Add job applications with company name, position, status and date.
- View all applications in a clean, tabular format.
- Edit existing entries directly from the table.
- Delete applications when they are no longer needed.
- Colored status labels for quick visual scanning.

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)

## How to Run Locally

1. Clone the repository:
git clone https://github.com/milanjyotiray/Job-Application-Tracker-HTML-CSS-JS-.git

2. Open the project folder.
3. Open `index.html` in your browser (or use a simple live server extension).

## Project Structure

- `index.html` – Main page with form and table layout.
- `style.css` – Styling for layout, table and status colours.
- `script.js` – Logic for adding, editing and deleting job applications.

## Possible Improvements
